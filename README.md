## Overview
A **Book Shop application** built in Java with FXML, Hibernate, and SpringBoot as a university course project.  
Users can browse, purchase, and comment on products, while managers can manage inventory and monitor registered users.  

---

## Features

### Customer
- Register and login  
- Browse products by category (Books, Stationery, Magazines)  
- Add/remove products to/from order list  
- Purchase products and save order history  
- Leave comments on purchased products  

### Manager
- Create, update, delete products  
- Monitor warehouse stock and user accounts  
- Manage product categories with field restrictions  

---

## Tech Stack
- **Backend:** Java, SpringBoot, Hibernate  
- **Frontend/UI:** Java + FXML  
- **Database:** Relational database (MySQL/PostgreSQL)  
- **Other libraries:** Lombok, Json
